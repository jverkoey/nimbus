Version 0.6 API Changes

Version 0.6.1 of Nimbus was released on August 8, 2011.

This minor version introduced the new Nimbus WebController, a ported version of Three20's TTWebController.

Added Frameworks

  • Nimbus WebController
Screenshot of a basic web controller on the iPhone

Noteworthy Non-API Changes

  • Added the Three20 lint tool. (thanks to rogchap.)
  • Added migration information from TTWebController to NIWebController. (thanks to rogchap.)
  • LICENSE and NOTICE files have been added to the project.
