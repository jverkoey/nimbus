Version 0.9 API Changes
Version History

Version 0.9.0 of Nimbus was released on October 24, 2011.

This major version introduced the new Nimbus CSS and Chameleon, a new way to rapidly prototype styling your iOS applications using CSS.

Watch the Chameleon Youtube video: http://www.youtube.com/watch?v=i_5LbQ8e9BU

Read the Chameleon blog post: http://blog.jeffverkoeyen.com/nimbus-chameleon
