00109 /*
00110   JKParseOptionComments        : Allow C style // and /_* ... *_/ (without a _, obviously) comments in JSON.
00111   JKParseOptionUnicodeNewlines : Allow Unicode recommended (?:\r\n|[\n\v\f\r\x85\p{Zl}\p{Zp}]) newlines.
00112   JKParseOptionLooseUnicode    : Normally the decoder will stop with an error at any malformed Unicode.
00113                                  This option allows JSON with malformed Unicode to be parsed without reporting an error.
00114                                  Any malformed Unicode is replaced with \uFFFD, or "REPLACEMENT CHARACTER".
00115  */
